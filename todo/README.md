# To-Do List Application

**シンプルで使いやすいタスク管理アプリケーション**

## 📋 機能

### コア機能
- ✅ タスク追加・削除・完了状態の切り替え
- 💾 **ローカルストレージで自動保存** - ページをリロードしても データが保持されます
- 🔍 フィルター機能（すべて・未完了・完了済み）
- 📊 リアルタイム統計情報表示
- 🎨 モダンでレスポンシブなデザイン
- ♿ XSS対策済みで安全

### UI/UX
- グラデーション背景でモダンな外観
- スムーズなアニメーション効果
- 直感的な操作性
- 📱 完全なモバイル対応（480px以上）
- ダークモード対応可能

## 🚀 使い方

### タスク追加
1. テキスト入力欄に新しいタスクを入力
2. 「追加」ボタンをクリック、または Enter キーを押す

### タスク管理
- ✓ **チェック**: 完了/未完了を切り替え
- **削除**: 削除ボタンで個別削除
- **完了済みを削除**: すべての完了済みタスクを一括削除

### フィルタリング
- **すべて**: すべてのタスクを表示
- **未完了**: 未完了タスクのみ表示
- **完了済み**: 完了済みタスクのみ表示

## 💾 データ保存

**ローカルストレージを使用**
- ブラウザの `localStorage` に自動保存
- ページをリロードしてもデータが保持される
- ブラウザのキャッシュクリアで削除される
- データ容量制限: 約 5-10MB（ブラウザによる）

### 保存データ
```json
{
  "id": 1234567890,
  "text": "タスク内容",
  "completed": false,
  "createdAt": "2026/7/14 10:30:45"
}
```

## 🔒 セキュリティ

- XSS（クロスサイトスクリプティング）対策済み
- HTMLエスケープ処理で安全にテキストを表示
- 入力値のバリデーション

## 📊 統計情報

リアルタイムで以下の情報を表示
- **総タスク数**: すべてのタスク数
- **未完了**: 完了していないタスク数
- **完了済み**: 完了済みのタスク数

## 🎨 カスタマイズ

スタイルは `styles.css` で変更可能
- カラー: `#667eea`, `#764ba2` などを変更
- フォント: `font-family` を修正
- 間隔: `padding`, `margin` を調整
